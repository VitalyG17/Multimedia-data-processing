Коэффициент управляет общим уровнем яркости изображения.
  • c > 1 увеличивает яркость, делая изображение светлее.
  • c < 1 уменьшает яркость, делая изображение темнее.
  • c = 1 не меняет яркость изображения.

Влияние гаммы

Гамма-коррекция влияет на контрастность изображения.
  • γ > 1 увеличивает контрастность, делая светлые области светлее, а темные - темнее.
  • γ < 1 уменьшает контрастность, делая изображение более плоским.
  • γ = 1 не меняет контрастность изображения.
Данный метод позволяет удалить из изображения пиксели, яркость которых выходит за заданный диапазон.
  • Мин. яркость - минимальное значение яркости пикселя, которое должно быть сохранено.
  • Макс. яркость - максимальное значение яркости пикселя, которое должно быть сохранено.
  • Все пиксели, яркость которых меньше минимального значения, заменяются на минимальную яркость.
  • Все пиксели, яркость которых больше максимального значения, заменяются на максимальную яркость.
